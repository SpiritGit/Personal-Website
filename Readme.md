# VuePress 开发静态博客

## 预览

**点击 [静态博客](https://spiritgit.github.io/blog-vuepress/) 查看线上效果**

## 本地如何运行
- 第一步：安装依赖文件
  执行 npm install

- 第二步：在本地起一个node服务
  执行 npm run dev

默认启用8080端口
访问 http://localhost:8080 即可

## 如何上线到github

执行 npm run deploy 即可更新到github

## 如何写文章
文章放置到_posts目录下

命名格式 xxxx-xx-xx-文章名字.md即可
